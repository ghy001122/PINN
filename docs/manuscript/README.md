# Manuscript versions

`main_submission_v1.md` is an immutable historical snapshot and is superseded for the current submission-lock assembly. `main_submission_v2.md` is the authoritative technical draft; its submission status remains fail-closed until the detached validation record is committed.

Version status is recorded in `manuscript_version_registry.md`; scientific claim authority remains `docs/paper/final_claim_matrix.md`.
