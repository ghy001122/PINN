# Data and code availability

The repository versions source code, configurations, tests, lightweight
JSON/CSV evidence, reports, claim maps, and the six main-figure PNG files.
Project results are synthetic numerical digital-twin outputs; no measured
device dataset is included.

Frozen Ground Truth v1.1 NPZ arrays are maintained as a hash-verified local
asset pack rather than redistributed in the repository. Publisher PDFs used
for source-contract auditing are third-party copyrighted materials, are
hash-locked locally, and are not redistributed. The associated provenance,
formula contracts, and derived numerical summaries are versioned.

Subject to repository policy and third-party licensing, the frozen benchmark
asset pack can be made available on reasonable request. A public clone alone
is therefore not represented as a complete bit-for-bit replay environment.
This boundary is deliberate and is checked by the submission-readiness audit.
