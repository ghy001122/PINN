# Conclusion V1

This work supports a calibration-gated sparse-port reduced inversion workflow for a synthetic phase-transition / memristive digital twin. The evidence shows that unrestricted hidden-field recovery from port data is ill-posed, while \(\gamma_{\mathrm{sub}}\) can be treated as a conditionally identifiable reduced target after \(T_{\mathrm{sw}}\) and related nuisance parameters are fixed or tightly bounded.

The manuscript should claim identifiability-guided target reduction and constrained thermal-parameter inversion in a one-dimensional synthetic benchmark. It should not claim experimental validation, full hidden-field recovery, or quasi-2D inverse diagnosis.
