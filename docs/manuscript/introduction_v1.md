# Introduction V1

Sparse terminal measurements are attractive for diagnosing oxide phase-transition and memristive devices, but terminal current \(I(t)\) and conductance \(G(t)\) compress electro-thermal-defect fields into low-dimensional port observables. This compression makes unrestricted hidden-field inversion underdetermined.

This work therefore frames the problem as identifiability-guided target-space reduction. Instead of claiming recovery of \(T(x,t)\), \(c_v(x,t)\), \(m(x,t)\), and \(\sigma(x,t)\) from port data alone, the manuscript focuses on a reduced scalar target, \(\gamma_{\mathrm{sub}}\), after literature-guided priors constrain micro-kinetic and switching parameters.

All data in this repository are synthetic numerical digital-twin benchmarks. They are not measured device data. The hardware motivation is a reduced phase-transition / memristive channel inspired by oxide devices, not a direct replication of any fabricated stack.
