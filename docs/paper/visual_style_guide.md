# Visual Style Guide

## Scope

Figures must be readable as synthetic numerical digital-twin benchmark evidence. Do not style or caption them as experimental device measurements.

## Defaults

- Use `matplotlib` only.
- Prefer white background, restrained colors, and visible axis labels with units.
- Put claim-boundary wording in captions, not decorative labels.
- Use relative error thresholds as dashed reference lines when relevant.
- Avoid overloaded legends; split figures if more than six curves compete.

## Main Figure Priorities

1. One main result per panel.
2. Keep `T_sw` confounding visually explicit.
3. Mark response-surface figures as interpolated/screening evidence.
4. Route F-SPS-PINN checks to appendix unless a future method paper is opened.
